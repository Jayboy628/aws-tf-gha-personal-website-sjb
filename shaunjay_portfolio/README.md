# Shaunjay J. Brown — Professional Portfolio

A responsive HTML, CSS, and JavaScript portfolio website designed for technology-leadership and data-engineering opportunities.

## Run locally

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Main files

- `index.html` — page content and portfolio sections
- `styles.css` — responsive design and visual styling
- `script.js` — mobile navigation, animations, and placeholder-link notices
- `assets/shaunjay-brown.jpg` — profile image
- `assets/healthcare-staffing-gaps.png` — healthcare case-study image
- `assets/healthcare-staffing-pain-points.png` — staffing pain-points diagram

## Before publishing

1. Replace all links marked with `data-placeholder-link` with real GitHub and video URLs.
2. Replace illustrative case-study outcomes with verified metrics or explicitly label them as targets.
3. Confirm company names, job dates, education, certifications, and contact details.
4. Replace placeholder project descriptions with your own problem, role, architecture, implementation, and measurable results.

## Publishing options

This static website can be hosted through GitHub Pages, Netlify, Cloudflare Pages, AWS Amplify, or an S3/CloudFront website.
